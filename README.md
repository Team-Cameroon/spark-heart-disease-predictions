
# 🫀 SPARK 2026 – Heart Disease Prediction Challenge

## 👥 Team Name: Team Cameroon 

---

## 📌 Project Overview
This project is part of the **SPARK Academy 2026 Mini Challenge**.

Our objective is to build a **machine learning model** that predicts whether a patient has heart disease using clinical data.

- Target:
  - `0` → No heart disease
  - `1` → Heart disease

- Problem Type:
  - Supervised Learning
  - Binary Classification

---

## 📊 Evaluation Metric
We are evaluated using **Macro F1-Score**, which balances performance across both classes.

---

## 📁 Dataset
The dataset is provided via Kaggle:
- `train.csv` → training data
- `test.csv` → prediction data
- `sample_submission.csv` → submission format

⚠️ Do NOT upload dataset files to this repository.  
Each member should download them from the whatsapp group individually.
